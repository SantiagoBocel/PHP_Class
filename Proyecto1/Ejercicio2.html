ETIQUETAS EN HTML

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una
persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página
web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo
mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.

Cuando escribes código en HTML, estás escribiendo etiquetas HTML. Todas las etiquetas HTML
están hechas con un número de partes específicas, incluyendo:

•El carácter “menor que” <
•Una palabra o carácter que determina qué etiqueta se está escribiendo
•Cualquier número de atributos HTML que se quiera usar, escritos de la forma nombre=” valor”
•El carácter “mayor que” >

<HTML>

Indicar el lenguaje que estamos trabajando, en este caso HTML.

<HEAD>

Indicar elementos que no seran visibles en la pagina web, pero que nos serviran para la interaccion. Para el buen funcionamiento.

<BODY>

Es el cuerpo de la pagina, lo que sera visible.

<H1...H6>

Son encabezados, dependiendo del numero, es el nivel que tendar.

<P>

Elementos tomados como parrafos dentro de la pagina.

<BR/>

Genera un salto de linea, no tiene etiqueta de cierre

<B>

Para colocar parrafos en negrita.

<I>

Para colocar cursiva en los parrafos

<U>

Para colocar subrayados en el parrafo

<STRONG>

Para indicar parrafos importantes. Tiene importancia semantica aumentada.

<SUB>

Para agregar subindices

<SUP>

Agregar superindices.

<MARK>

Para marcar texto dentro de nuestro parrafo.

<HR/>

Para dividir los parrafos con una linea. No tiene etiqueta de cierre.
